# API Garagem - Gerenciador de Carros Clássicos

API REST para gerenciar uma coleção de carros clássicos em uma garagem.

## Tecnologias

- Node.js
- Express.js
- Sequelize (ORM)
- SQLite (Banco de dados)

## Como rodar o projeto

1. Clone o repositório:
```bash
git clone https://github.com/wesleyafonsogomes/garagem-api.git
```

2. Instale as dependências:
```bash
npm install express sequelize sqlite3 
```

3. Inicie o servidor:
```bash
node app.js
```

4. Acesse: `http://localhost:3005`
